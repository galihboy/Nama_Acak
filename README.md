## Pembangkit Nama Acak
###### Developed by [Galih Hermawan](https://galih.eu) | [Blog Medium](https://masgalih.medium.com).

Cek demo online di https://galihboy.github.io/mini_projects/

```
nama_islam.txt
nama_laki.txt
nama_nabi.txt
nama_sahabat_nabi.txt
```
